# 🐍 Snake Game - Multi-Language Edition

Dos versiones del clásico juego de Snake programadas en diferentes lenguajes.

## 📁 Estructura del Proyecto

```
snake-gama/
├── README.md                  # Este archivo
├── javascript/               # Versión en JavaScript
│   ├── index.html           # HTML con estilos integrados
│   ├── script.js            # Lógica del juego
│   └── README.md            # Documentación JavaScript
└── python/                   # Versión en Python
    ├── snake_game.py        # Archivo principal
    ├── requirements.txt     # Dependencias
    └── README.md           # Documentación Python
```

## 🎮 Versiones Disponibles

### 🌐 Versión JavaScript
Una versión web interactiva del juego de Snake con controles por teclado y táctiles.

**Características:**
- Juega directamente en el navegador
- Controles: Flechas de dirección, teclado o touch
- 4 niveles de dificultad
- Gráficos suave con Canvas

**Para jugar:** Abre `javascript/index.html` en tu navegador

---

### 🐍 Versión Python
Una versión de escritorio del juego con gráficos mejorados usando Pygame.

**Características:**
- Gráficos 2D con Pygame
- Sistema de dificultad progresivo
- Almacenamiento de puntuaciones
- Cuadrícula visual

**Para jugar:**
```bash
cd python
pip install -r requirements.txt
python snake_game.py
```

---

## 🎯 Cómo Jugar

### ⌨️ Controles Generales

| Acción | JavaScript | Python |
|--------|-----------|--------|
| Mover arriba | ⬆️ Flecha | ⬆️ Flecha |
| Mover abajo | ⬇️ Flecha | ⬇️ Flecha |
| Mover izquierda | ⬅️ Flecha | ⬅️ Flecha |
| Mover derecha | ➡️ Flecha | ➡️ Flecha |
| Iniciar/Pausar | Click "Start" | ESPACIO |
| Reiniciar | Click "Restart" | R |

### 🎯 Objetivo
Controla la serpiente para que coma la comida roja y crezca. Evita chocar con las paredes y contigo mismo.

---

## 📊 Sistema de Puntuación

- **+10 puntos** por cada comida comida
- **Dificultad aumenta** automáticamente
- **Mejor puntuación** se guarda
